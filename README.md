# Cerebro-sinonimos

> A Cerebro plugin to look up synonyms of a word (pt-BR Only)

![](screenshot.png)

## Usage
Type any word you'd like to get some synonyms for!

## Features

* Get a set of synonyms grouped by similar meanings.
